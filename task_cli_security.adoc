= 變更部署管理員密碼

您可以使用命令列介面、視需要變更部署虛擬機器管理員帳戶的密碼。

.步驟
. 使用系統管理員帳戶登入部署公用程式CLI。
. 變更密碼：「密碼修改」
. 根據您的環境回應所有提示。
